When you start exposing the sensor array, each pixel starts measuring RGB intensity,

When light hits a pixel, the intensity being recorded increases. Intensity is measured from 0 (black) to 255 (the max intensity for that color) until you either stop exposing the sensor array or until the pixel reaches the max value
