- to store the color for each pixel
- pixel has three color channels, red, green, and blue (RGB)
- each channel has a value between 0 and 255
- the greater the value, the more intense the color
- these three channels combine to form the final color of the pixel

- for example, a pixel with RGB values of 255, 255, and 255 would be white
- and a pixel with RGB values of 0, 0, and 0 would be black
