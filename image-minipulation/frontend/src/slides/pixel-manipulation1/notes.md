- one thing we can do to manipulate images is by changing the color values of individual pixels
- for example, we can increase and decrease the brightness
- in the same way your phone's image editing apps do

- to increase the brightness, for every pixel in the image, we can add a value to all three color channels
- as you can see in the middle, the image becomes brighter

- to decrease the brightness, we can subtract a value from all three color channels
- as you can see on the right, the image becomes darker
