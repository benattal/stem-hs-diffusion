## What is Noise?

- Explain: **noise** is just random pixel values — imagine every pixel in an image getting a random color. That's it!
- Show the slider demo: start with a clean image on the left and drag to the right to see noise being added
- Point out: at the far right, the image is completely unrecognizable — it's pure random noise
- Try different images to show it works the same way regardless of the content
- Key question: "What if we could run this process in reverse? Start with noise and end up with a real image?"
- That's exactly what a diffusion model does — it learns to undo the noise, step by step
